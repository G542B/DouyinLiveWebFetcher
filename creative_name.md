## 创意名称

**🎯 弹幕智趣 - AI驱动的直播互动引擎**

## 创意介绍

### 一句话简介

基于 AI 语义理解技术的抖音直播弹幕智能互动系统，打造沉浸式直播互动体验。

### 详细介绍

随着直播行业的快速发展，弹幕已成为观众互动的核心方式。然而，传统弹幕系统面临信息过载、互动形式单一、缺乏语义理解等痛点。

本项目创新性地将 AI 语义理解技术融入直播弹幕系统，构建了一个智能互动平台：

**核心创新点：**

1. **语义级弹幕过滤**：利用 bge-small-zh-v1.5 Embedding 模型实现语义相似度匹配，精准过滤无效信息，提升内容质量。
2. **AI 猜词互动游戏**：观众通过弹幕发送猜测，系统实时计算猜测与目标词的语义相似度，实现智能化评分，增强直播趣味性和观众参与感。
3. **实时智能排行榜**：基于语义相似度评分的实时排名系统，激发观众竞争热情，打造活跃的直播间氛围。
4. **多端部署支持**：支持 Web、桌面（Electron）等多种部署方式，满足不同场景需求。

**技术亮点：**

- 智能语义分析：集成 Embedding 模型，实现精准中文语义相似度计算
- 高性能消息处理：基于队列的消息广播机制，支持高并发弹幕处理
- 模块化架构：前后端分离，支持独立部署和扩展
- 自动化输出：集成 Playwright 实现浏览器自动化弹幕输出

本项目致力于推动直播互动从"被动观看"向"主动参与"转变，为直播生态注入智能化活力。

## 技术栈

- **前端**：Vue 3 + Element Plus + Vite
- **后端**：FastAPI + Python 3.11
- **AI 模型**：bge-small-zh-v1.5（Embedding）
- **部署**：Electron（桌面）+ Web（浏览器）
- **消息通信**：WebSocket
- **自动化**：Playwright

***

## 适用场景

1. **直播互动**：主播发起猜词游戏，观众通过弹幕参与
2. **内容过滤**：过滤无效弹幕，提升直播间内容质量
3. **数据采集**：弹幕数据实时采集与分析
4. **二次创作**：弹幕数据导出用于内容创作
